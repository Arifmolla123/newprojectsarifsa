কম্পিউটার কিবোর্ড শর্টকার্ট টুল

Win + E

File Explorer খুলে।

Win + L

কম্পিউটার লক করে দেয়।

Ctrl + D

বর্তমান ওয়েবপেজ বুকমার্ক করে।

Ctrl + Shift + N

Incognito / Private tab খোলে।

Alt + F4

বর্তমান অ্যাপ বা উইন্ডো বন্ধ করে।

F5

রিফ্রেশ করে (Page বা Folder)।

Ctrl + P

প্রিন্ট অপশন খোলে।

Ctrl + S

ডকুমেন্ট বা কাজ সেভ করে।

Ctrl + Shift + Esc

Direct Task Manager খুলে।

Ctrl + Z

শেষ কাজ Undo করে।

Ctrl + Y

Undo করা কাজ Redo করে।

Ctrl + A

সব কিছু সিলেক্ট করে।

Ctrl + C

কপি করে।

Ctrl + V

পেস্ট করে।

Ctrl + X

কাট করে।

Ctrl + F

পেজে কিছু সার্চ করার জন্য।
