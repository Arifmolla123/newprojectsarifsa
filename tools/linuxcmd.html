💻 Linux Commands CheatSheet

Made with ❤️ by Arif

📍 pwd

বর্তমান অবস্থান (ডিরেক্টরি) দেখায়।

📁 mkdir

নতুন ফোল্ডার তৈরি করে।

🔀 mv

ফাইল মুভ বা রিনেম করতে ব্যবহার হয়।

📄 cp

ফাইল কপি করতে ব্যবহৃত হয়।

💽 df -h

ডিস্কের স্পেসের তথ্য দেয়।

📊 top

রিয়েল-টাইমে সিস্টেম প্রসেস দেখায়।

🔎 ps aux

চলমান প্রসেসসমূহ দেখায়।

❌ kill

প্রসেস বন্ধ করে (PID অনুযায়ী)।

📘 man

কমান্ডের ম্যানুয়াল দেখায়।

👤 whoami

বর্তমান ইউজার দেখায়।

🌐 hostname

সিস্টেমের হোস্টনেম দেখায়।

📶 ping

ইন্টারনেট কানেকশন চেক করতে ব্যবহৃত হয়।

🧹 clear

টার্মিনাল স্ক্রিন পরিষ্কার করে।

🛑 shutdown

সিস্টেম বন্ধ করে দেয়।

📂 ls -la

সব ফাইল ও হিডেন ফাইল লিস্ট দেখায় (ডিটেইল সহ)।

📁 cd /path

নির্দিষ্ট লোকেশনে যায়।

📄 cat

ফাইলের কনটেন্ট দেখায়।

✏️ nano

ফাইল এডিট করার জন্য সহজ টেক্সট এডিটর।

🔐 chmod +x

ফাইলকে executable করে তোলে।

📦 sudo apt update

প্যাকেজ লিস্ট আপডেট করে।

⬆️ sudo apt upgrade

সকল প্যাকেজ আপগ্রেড করে।

📥 apt install

নতুন সফটওয়্যার ইনস্টল করে।

❌ apt remove

সফটওয়্যার আনইনস্টল করে।

📜 history

ব্যবহৃত আগের কমান্ডগুলো দেখায়।

🔍 grep

ফাইলে নির্দিষ্ট শব্দ খোঁজে।

💾 df -h

ডিস্ক ইউসেজ দেখায় (human readable)।

📦 du -sh

ফোল্ডারের সাইজ দেখায়।

🧠 uname -a

সিস্টেমের তথ্য দেখায়।

🌐 ifconfig

নেটওয়ার্ক ইন্টারফেসের ডিটেইল।

🔌 netstat

ওপেন পোর্ট এবং কানেকশন দেখায়।

💽 df -Th

ডিস্কের ধরন (Type) ও ইউসেজ দেখায় (human readable)।

⏱️ uptime

সিস্টেম কতক্ষণ চালু আছে দেখায়।

🔁 reboot

সিস্টেম রিস্টার্ট করে।

⛔ shutdown now

সিস্টেম সঙ্গে সঙ্গে বন্ধ করে।

📜 history | grep

সুনির্দিষ্ট কমান্ড সার্চ করে আগের history থেকে।

🔍 find

পুরো সিস্টেমে ফাইল খোঁজে।

📁 locate

দ্রুত ফাইল খুঁজে (updatedb ইনডেক্স দরকার)।

🔄 updatedb

locate কমান্ডের জন্য ফাইল ইনডেক্স আপডেট করে।

🔐 chmod 755

ফাইল পারমিশন সেট করে (owner rwx, group/others rx)।

👤 chown

ফাইলের মালিকানা বদলায়।

➕ adduser

নতুন ইউজার তৈরি করে।

🔑 passwd

ইউজারের পাসওয়ার্ড সেট/পরিবর্তন করে।

⏰ crontab -e

শিডিউল কাজের জন্য Cron Jobs সেট করে।

🚀 service start

সার্ভিস চালু করে (যেমন: apache2)।

📊 systemctl status

কোনো সার্ভিসের স্ট্যাটাস দেখে।
